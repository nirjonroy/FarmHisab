# FarmHisab Project Plan

FarmHisab will be built in planned phases. This foundation step prepares the Laravel project, frontend asset pipeline, documentation, and future API authentication base without creating business modules yet.

## Planned Modules

- Authentication and user roles
- Farms and sheds
- Bird types and breeds
- Batch management
- Daily farm records
- Mortality tracking
- Weight tracking
- Feed purchase and usage
- Medicine and vaccination
- Expenses
- Sales
- Suppliers and buyers
- Inventory
- Cashbook and due management
- Profit and loss reports
- Notifications
- Android REST API
